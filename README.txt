----- WELCOME TO SAMPLER TOOL 							by Rioko & Miso		----
-----													----
------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------

This Git Repository give you free access to the code necessary to modify and test the SAMPLER plug-in in 
Eclipse. SAMPLER is a plug-in designed to draw EMF models and navigate through them. It also allows the user
to search an object of the model, or reorganize the view with different layouts.

In order to execute the tool, it is necessary clone the four projects in this repository, or at least the 
drawModels, graphAbstraction and plugin.utils.

Enjoy SAMPLER.

----- BIENVENIDOS A SAMPLER							by Rioko & Miso		----
-----													----
------------------------------------------------------------------------------------------------------------
------------------------------------------------------------------------------------------------------------

Este repositorio de Git da libre acceso al código necesario para modificar y probar la herramienta SAMPLER
de Eclipse. SAMPLER es un plug-in diseñado para visualizar y navegar modelos EMF. También permite al usuario
buscar un objeto concreto del modelo y reorganizar la visualización con diferentes layouts.

Para ejecutar la herramienta, es necesario clonar los cuatro proyectos de este repositorio; o al menos las 
carpetas drawModels, graphAbstraction y plugin.utils.

¡Disfrutad de SAMPLER!
